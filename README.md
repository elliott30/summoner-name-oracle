# Summoner Name Oracle

> *For when "xXDarkLord99Xx" is already taken (again)*

A League of Legends summoner name generator that crafts funny, thematic, and Riot-compliant usernames using 16 clever algorithmic patterns and a deep pool of champions, adjectives, and gaming slang.

## What it does

Paste-worthy summoner names in one click. Each generated name:

- Stays within Riot's 3–16 character limit
- Comes with a Riot ID tag (e.g. `#EUW`, `#TILT`, `#FF15`)
- Shows the pattern that created it and a humorous quip
- Can be copied as a standalone name or full Riot ID
- Keeps a history of your last 9 generated names

With 168 champions, 220+ adjectives, and 16 generation patterns, there are over 18 million possible name combinations.

## Generation patterns

The 16 patterns include classics like *The Struggle* (`hardstuckAhri`), *The Accusation* (`ZedIsFeeding`), *The Confessional* (`ieatwardsDaily`), *The Humble* (`notAZedMain`), and more.

## Stack

| Layer | Technology |
|---|---|
| Framework | [Next.js](https://nextjs.org/) 15 |
| UI | React 18 |
| Styling | CSS-in-JS (styled-jsx) + Google Fonts (Cinzel, Open Sans) |
| Logic | Vanilla JavaScript — no external dependencies |
| Data | Hardcoded champion list, adjective bank, and pattern library |
| Deployment | Vercel-ready |

No backend, no database, no API keys. Everything runs client-side.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
